1. What is Linux?

Linux is a free and open-source operating system based on the Unix operating system. It acts as an interface between the user and computer hardware and is widely used in servers, desktops, mobile devices, and embedded systems.

Features of Linux:

Open source and free to use
Multi-user operating system
Multitasking support
Secure and stable
Portable across different hardware platforms
